# Security Policy

## About This Repository

This repository is a **reproducibility artifact** accompanying the published research paper:

> Campbell, R. *[Parameter-Resident Cryptographic Material as an Unscoped Surface for Post-Quantum Migration: An Existence Proof and Audit Primitive]*. [MDPI Computers], [2026]. DOI: [10.20944/preprints202605.0601.v1]

It contains research code, experimental scripts, and configuration files used to produce the results reported in the paper. It is **not** production cryptographic software and is **not** intended for use in security-sensitive systems, deployed applications, or any setting where the confidentiality, integrity, or availability of real data depends on its correctness.

## Intended Use

This code is intended for:

- Independent verification of the results reported in the accompanying paper
- Academic study and peer review
- Follow-on research that builds on or critiques the published methodology

This code is **not** intended for, and must not be used for:

- Protection of production data
- Cryptographic operations in deployed systems
- Any application where a failure of the cryptographic implementation would cause harm

## Reporting Concerns

This is a frozen artifact: the code as published is the code that produced the results in the paper. Modifications would break reproducibility against the published results, so pull requests altering the code will not be accepted.
